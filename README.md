# Star-Rush
 Durante uma viagem espacial, o foguete de dois exploradores cai em um planeta desconhecido. Presos no local, eles enfrentam aliens que dominam a região. A única forma de sair do planeta é derrotar os extraterrestres e consertar a nave antes que o tempo acabe.
 Para isso, os jogadores precisam explorar o planeta, coletar os itens necessários espalhados pelo cenário e enfrentar os inimigos até concluir o concerto da nave.
